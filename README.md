# Python-Basics

This repository contains exercises and examples of **basic Python concepts** created by AmirKhan.  
Goal: To summarize the fundamental concepts of Python and prepare for real projects and practical exercises.

## Folder Structure

1. **01_Variables** - Variables and data types  
2. **02_Strings** - Strings, slicing, formatting, and common string methods  
3. **03_Exceptions** - Error handling with try-except, raise, finally  
4. **04_Lists_Sets_Dicts** - Lists, loops, Sets,tuple and Dictionaries  
5. **05_Files_JSON** - Working with files and saving/loading data with JSON

## Notes
- Each file contains practical examples and exercises  
- File and folder names are organized according to the lessons
